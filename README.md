# Система прогнозирования спроса

Прототип аналитической системы для прогнозирования спроса с иерархическим согласованием прогнозов.

## Описание

Приложение для автоматизации построения прогнозов на разных уровнях иерархии (SKU, категория, магазин), оценки их точности и выявления проблемных зон.

## Функциональность

- Загрузка и предобработка данных M5 Forecasting
- Анализ временных рядов: определение тренда и сезонности
- XYZ/ABC-анализ для сегментации товаров
- Иерархическое прогнозирование (Bottom-up, Top-down)
- Статистические модели (ARIMA, ETS) и ML-подходы (LightGBM, CatBoost)
- Интерактивный дашборд с визуализацией
- Метрики качества: WAPE, MAE, RMSE
- Exception Management: выявление проблемных товаров

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

Приложение будет доступно по адресу: `http://localhost:8501`
